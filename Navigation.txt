HE ls
    - We can enter ls to get a list of files and subdirectories contained inthe current working directory.

    - Beside the current diorectory, we can specify the directory to list.
        >> ls /usr

    - We can even specify multiple directories
        >> ls ~ /usr

    - We can also change the fpormat of the output to reveal more detail.
        >> ls -l
        >> la -a
